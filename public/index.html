Get Started

Jump right into building with Picco—use the CDN, install it via package manager, or download the source code.

Download (4kB)

Install via package manager

Install Picco source css files via npm. Package managed installs don’t include documentation or our full build scripts.

Install via npm:

npm install piccocss

Include via CDN

When you only need to include Picco compiled CSS, you can use jsDelivr. See it in action with our simple quick start.

Include this in your html head: 

<link rel="stylesheet" href="PACKAGE_URL">

Native dark theme

All the styles needed are build in and customizable, all do you need is switch a class

Use this example in your html file for see dark theme just change:

<a href="javascript:void(0)" onclick="() => 
    (document.body.classList.contains('dark')) ? document.body.classList.remove('dark') : document.body.classList.add('dark')">
    Switch Theme
</a>
