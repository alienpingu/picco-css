Tab

The Tab component is used to create tabbed navigation interfaces. It allows you to organize content into different sections, each accessible by clicking on a tab.

Tab 1 Tab 2 Tab 3 
<div class="tabs">
    <a href="#tab1" aria-current="page">Tab 1</a>
    <a href="#tab2">Tab 2</a>
    <a href="#tab3">Tab 3</a>
</div>
